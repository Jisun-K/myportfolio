# 🌐 Portfolio Website

개인 포트폴리오 웹사이트입니다.  
저의 프로젝트와 경험을 정리하고, React/Next.js 기반의 개발 역량을 보여주기 위해 제작했습니다.

---

## 🚀 기술 스택
- **Framework**: Next.js
- **UI 라이브러리**: React, Tailwind CSS, shadcn/ui
<!-- - **상태 관리**: Zustand
- **배포**: Vercel -->

---

## ✨ 주요 기능
- **프로젝트 섹션**: Work / Personal 프로젝트를 구분해 소개
- **경력 소개**: 연도별 경험 정리
- **스킬 차트**: 숙련도를 시각적으로 표현
- **반응형 디자인**: 다양한 디바이스에서 최적화된 UI 제공
